# 🚀 Limitless Web Platform

A full-stack web application built with **Symfony 5** that integrates modern web features such as **Stripe payments, QR code generation, Google Charts analytics, reCAPTCHA security, and PDF generation**.

This project demonstrates a complete web platform architecture using Symfony, Doctrine ORM, and modern PHP development practices.

---

# 📌 Features

- 💳 **Stripe Payment Integration**
- 📊 **Data Visualization with Google Charts**
- 🔐 **Google reCAPTCHA v3 Protection**
- 📄 **PDF Generation with DomPDF**
- 📱 **QR Code Generation**
- 📑 **Pagination System (KNP Paginator)**
- 🗄 **Database Management with Doctrine ORM**
- 🔄 **Doctrine Migrations**
- 🧠 **MVC Architecture using Symfony Framework**
